# 🔨 u-row组件可选配置的参数

属性 可选参数 默认值 类型 说明
warp true / false false string 是否开启自适应,说明:设置了此属性后row内部的col或者其他元素会进行自动编排
justify-start true / false false string 布局对齐 左对齐
justify-end true / false false string 布局对齐 右对齐
justify-eenter true / false false string 布局对齐 居中对齐
justify-between true / false false string 布局对齐 左右两边对齐
justify-around true / false false string 布局对齐 各放置左右两边的正中位置
align-start true / false false string 垂直布局的位置 上对齐
align-end true / false false string 垂直布局的位置 底部对齐
align-center true / false false string 垂直布局的位置 居中

WARNING

配置参数的justify布局参数只需要选择一个,请勿配置多个参数,这个参数唯一。 align-start 也同等效果 只需要配置一个 比如align-start 和 align-end align-center 只需要一个就可以了

# 🔗 u-col组件可选配置的参数

属性 可选参数 默认值 类型 说明
span 1 - 5 1 string col占用row的几行 1 - 5 1为20%,除3固定是50%外,其他配置的参数各 x 20%
