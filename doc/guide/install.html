    # 安装

    项目支持npm包管理方式导入项目中，安装完毕后 将项目依赖挂在到vue的main主路口文件中。

    npm install color-ui-next

    main.js / main.ts 主路口挂载

    import colorUiNext from 'color-ui-next'
import '@color-ui-next/main.css'
import '@color-ui-next/icon.css'
