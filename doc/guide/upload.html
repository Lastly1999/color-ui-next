# 🔍 Upload上传组件

TIP

一个上传组件，在原先的基础上加入了双向绑定，图片上传和删除绑定的值会同步改变，上传的方式为uni-app的uni.chooseImage api

<cu-upload :max="10" @delete="del" @change="onChangeUpload" v-model="value"/>

export default{
    data(){
        return{
            value:''
        }
    },
    methods:{
        del(item){
            // item 为删除照片的对象
            console.log(item)
        },
        onChangeUpload(item){
            // item 上传照片的对象
            console.log(item)
        }
    }
}

# 🔨 Upload可配置的参数

属性 可选参数 类型 说明
max 整数 number 最大可上传数量
v-model 空 string[] 上传的图片集合，双向绑定

# 🔗 Upload事件

事件名 说明 回调
@change 组件上传改变 上传照片的对象
@del 删除事件 删除照片的对象
