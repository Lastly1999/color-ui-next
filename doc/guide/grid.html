# 📰 Grid 宫格菜单

TIP

用来定义一些单元格菜单，导航菜单，首页菜单等场景的组件


<template>
    <cu-grid :data="data" @click="colClick"></cu-grid>
</template>

export default {
      data() {
            return {
                  data: []
            }
      },
      methods:{
            colClick(item){
                  // 返回参数见下方接口描述
                  console.log(item)
            }
      }
}

Grid的接口字段约束

cuIcon:单元格的图标
color:单元格的图标颜色
badge:单元格的角标数量
name:单元格的文本

interface DefaultProps {
    cuIcon: string
    color: string
    badge: string
    name: string
}
