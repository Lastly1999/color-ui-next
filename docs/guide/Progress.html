# 🔋 Progress 进度条

TIP

该组件一般用于记录进度和一些加载进度信息。

    默认进度条
    <cu-progress loading value="80"></cu-progress>
    中进度条
    <cu-progress loading size="sm" value="80" ></cu-progress>
    大进度条
    <cu-progress loading size="xs" value="80"></cu-progress>
    <!-- 进度条形状 -->
    默认
    <cu-progress bg="red" loading value="50">50%</cu-progress>
    圆弧
    <cu-progress bg="red" loading radius  value="70">70%</cu-progress>
    椭圆
    <cu-progress bg="red" loading round value="90">90%</cu-progress>
    <!-- 进度显示条纹 -->
    默认
    <cu-progress bg="yellow" loading striped value="50">50%</cu-progress>
    圆弧
    <cu-progress bg="yellow" loading radius striped  value="70">70%</cu-progress>
    椭圆
    <cu-progress bg="yellow" loading round striped value="90">90%</cu-progress>
    满进度
    <cu-progress bg="green" loading round striped value="100">100%</cu-progress>

# 🔗 进度条可选配置的参数

属性 可选参数 默认值 类型 说明
loading true / false true boolean 是否显示加载进度
bg 参考colorui颜色表 blue string 进度条百分比颜色
value 10 10 string 当前进度百分比：如百分之50value值就是50
size 默认 / sm / xs 默认 string 进度条尺寸
round true / false false boolean round扁平进度条
radius true / false false boolean radius圆形进度条
striped true / false false boolean 是否为头像的徽章
