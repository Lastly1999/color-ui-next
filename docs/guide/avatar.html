# 👹 Avatar 头像

TIP

基于color-ui二次封装的头像组件

      <!-- 圆形的头像 -->
      <cu-avatar round size="sm" src="https://ossweb-img.qq.com/images/lol/web201310/skin/big10001.jpg"></cu-avatar>
      <cu-avatar round src="https://ossweb-img.qq.com/images/lol/web201310/skin/big10001.jpg"></cu-avatar>
      <cu-avatar round size="lg" src="https://ossweb-img.qq.com/images/lol/web201310/skin/big10001.jpg"></cu-avatar>
      <cu-avatar round size="xl" src="https://ossweb-img.qq.com/images/lol/web201310/skin/big10001.jpg"></cu-avatar>
      <!-- 扁平的头像 -->
      <cu-avatar radius size="sm" src="https://ossweb-img.qq.com/images/lol/web201310/skin/big10001.jpg"></cu-avatar>
      <cu-avatar radius src="https://ossweb-img.qq.com/images/lol/web201310/skin/big10001.jpg"></cu-avatar>
      <cu-avatar radius size="lg" src="https://ossweb-img.qq.com/images/lol/web201310/skin/big10001.jpg"></cu-avatar>
      <cu-avatar radius size="xl" src="https://ossweb-img.qq.com/images/lol/web201310/skin/big10001.jpg"></cu-avatar>
      <!-- 头像组 -->
      <cu-avatar-group>
        <cu-avatar round size="xl" src="https://ossweb-img.qq.com/images/lol/web201310/skin/big10001.jpg"></cu-avatar>
        <cu-avatar round size="xl" src="https://ossweb-img.qq.com/images/lol/web201310/skin/big10001.jpg"></cu-avatar>
        <cu-avatar round size="xl" src="https://ossweb-img.qq.com/images/lol/web201310/skin/big10001.jpg"></cu-avatar>
        <cu-avatar round size="xl" src="https://ossweb-img.qq.com/images/lol/web201310/skin/big10001.jpg"></cu-avatar>
      </cu-avatar-group>
      <!-- 头像带标签 -->
      <cu-avatar round size="xl" src="https://ossweb-img.qq.com/images/lol/web201310/skin/big10001.jpg">
          <cu-tag badge bg="blue" type="radius">
                <cu-icon icon="male"></cu-icon>
          </cu-tag>
      </cu-avatar>
      <cu-avatar round size="xl" src="https://ossweb-img.qq.com/images/lol/web201310/skin/big10001.jpg">
          <cu-tag badge bg="pink">
            <cu-icon icon="female"></cu-icon>
          </cu-tag>
      </cu-avatar>

# 🔨 头像组件可选配置的参数

属性 可选参数 默认值 类型 说明
size sm / 默认 / lg / xl 默认 string 头像尺寸
src 头像地址 空 string 头像地址,base64或者图片地址
round true / false false string 头像形状,有参数时为圆形
radius true / false false string 头像形状,有参数时为扁平
bg 参考colorui颜色表 空 string 头像背景颜色

WARNING

如要使用头像组功能 请务必在头像组的组件外部添加头像组件 ：例

<cu-avatar-group>
        <cu-avatar round size="xl" src="https://ossweb-img.qq.com/images/lol/web201310/skin/big10001.jpg"></cu-avatar>
        <cu-avatar round size="xl" src="https://ossweb-img.qq.com/images/lol/web201310/skin/big10001.jpg"></cu-avatar>
        <cu-avatar round size="xl" src="https://ossweb-img.qq.com/images/lol/web201310/skin/big10001.jpg"></cu-avatar>
        <cu-avatar round size="xl" src="https://ossweb-img.qq.com/images/lol/web201310/skin/big10001.jpg"></cu-avatar>
</cu-avatar-group>
