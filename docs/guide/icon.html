# 🔗 Icon 图标

TIP

一些color-ui内置的二次封装图标组件。

    图标
    <cu-icon icon="female"></cu-icon>
    图标背景色
    <cu-icon icon="female" bg='blue'></cu-icon>

# 🔨 图标可选配置的参数

属性 可选参数 默认值 类型 说明
icon 参考colorui图标列表 空 string 图标的文本
bg 参考colorui颜色表 blue string 图标的背景颜色

# 🔗 图标事件

事件名 说明 回调
@click 图标的点击事件 无回调
