# 📋 TimeLine 时间轴组件

TIP

该组件一般用于流程进度的展示和预览


<template>
    <cu-timeline title="04/13" :data="data" @click="colClick"></cu-timeline>
</template>

export default {
      data() {
            return {
                  data: [
                        {
                              content: '【2021-03-02】您的快递已经打包出库',
                              icon: 'timefill',
                              iconColor: ''
                        },
                        {
                              content: '【2021-03-03】快递到达中转站',
                              icon: 'timefill',
                        },
                        {
                              content: '【2021-03-04】快递正在安排派送，快递小哥电话：19955223111，请保持电话畅通',
                              color: '',
                              icon: 'timefill',
                              iconColor: ''
                        },
                        {
                              content: '【2021-03-04】快递已经签收，签收人：老王。',
                              color: 'green',
                              icon: 'check'
                        }]
            }
      }
}

TimeLine的接口字段约束

interface DefaultProps {
    content: string
    color: string
    icon: string
}

# 🔨 TimeLine可配置的参数

属性 可选参数 类型 说明
data [] DefaultProps[] 动态循环的数据展示，dataCom模式,内对象见上方接口
title null string 每个TimeLine的标题文字
