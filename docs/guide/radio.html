# 🔴 Radio 单项选择器

TIP

该组件一般单选选项 适用于表单

<cu-radio color="green" v-model="value"/>

export default{
    data(){
        return{
            value:true
        }
    }
}

# 🎨 自定义Radio选中颜色

<cu-radio color="green" v-model="value"/>

在属性加入color=“颜色值” 即可变换颜色

# 🔨 Radio可选配置的参数

属性 可选参数 默认值 类型 说明
v-model any 空 any input绑定的值(双向绑定)
placeholder any 空 any input的输入提示信息
