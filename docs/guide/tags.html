# 📌 Tags 徽章

TIP

该组件一般用于信息标记和未读数量的消息提示。

   <cu-tag bg="blue">默认</cu-tag>
   <cu-tag bg="cyan" type="radius">普通尺寸</cu-tag>
   <cu-tag bg="red" size="sm" type="round">椭圆小尺寸</cu-tag>
   <cu-tag bg="cyan" line type="radius">镂空普通尺寸</cu-tag>
   <cu-tag bg="blue" line type="radius">普通尺寸</cu-tag>
   <!-- 自定义徽章文字颜色 -->
   <p>带图标的标签</p>
   <cu-tag bg="green" color="blue" type="radius">自定义文字颜色</cu-tag>
   <!-- 带图标的徽章 -->
   <p>带图标的标签</p>
   <cu-tag bg="green" color="blue" type="radius">
        <cu-icon icon="check">get</cu-icon>
   </cu-tag>
   <cu-tag bg="red" color="blue" type="radius">
        <cu-icon icon="likefill">loveyou</cu-icon>
   </cu-tag>
   <cu-tag bg="red" color="blue" type="radius">
        <cu-icon icon="appreciate">yeah</cu-icon>
   </cu-tag>

# 🔨 徽章可选配置的参数

属性 可选参数 默认值 类型 说明
size small / 默认 默认大小 string 徽章尺寸
bg 参考colorui颜色表 white string 徽章颜色
type round / radius blue string 徽章形状
line true / false false boolean 是否镂空
color 参考colorui颜色表 false boolean 徽章文字颜色
badge true / false false boolean 是否为头像的徽章
