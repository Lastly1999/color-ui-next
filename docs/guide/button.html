# 🆗 Button 按钮

TIP

组件库内置了ColorUI所有的按钮样式，使用时只需传入相对应的配置即可使用按钮

   <cu-button type="blue" @click="submit">按钮</cu-button>
   <cu-button type="red"  @click="submit">按钮</cu-button>
   <cu-button round type="red" @click="submit">圆形按钮</cu-button>
   <cu-button shadow  type="pink" @click="submit">阴影按钮</cu-button>

# 🔨 按钮可选配置的参数

属性 可选参数 默认值 类型 说明
size small / lg 默认大小 string 按钮尺寸
color 参考colorui颜色表 white string 按钮文字颜色
type 参考colorui颜色表 blue string 按钮类型
disabled true / false false boolean 按钮启用状态
line true / false false boolean 是否镂空按钮
round true / false false boolean 是否椭圆按钮
shadow true / false false boolean 是否显示阴影

# 🔗 按钮事件

事件名 说明 回调
@click 按钮点击事件 无回调
